# 高 `Npl` 尾部残差校正实施计划

> 目标不变：围绕“统一方法 / 统一框架 / 统一输入体系”完成多截面 CFST 柱承载力预测，而不是退回到按截面或工况拆成多个彼此独立的主模型。

## 1. 当前判断

- 在 `data/raw/2026.3.11-new.csv` 上，当前最优单模型为：
  - `target_mode: raw`
  - `target_transform: log`
  - 保留全部数值几何特征
- 当前 holdout 指标约为：
  - `R² = 0.9857`
  - `COV = 0.0976`
- 主瓶颈不是整体稳定性，而是高尺度尾部对平方误差的主导：
  - `scale_npl_q5` 贡献了大部分 `SSE`
  - 超大承载力样本数量少，但对 `R²` 拖累极大

因此，下一阶段不再优先做全局小幅调参，而是转向“全局模型 + 高 `Npl` 尾部残差校正”。

## 2. 方法定位

方法名称建议表述为：

- `scale-aware residual correction`
- 或 `global prediction with high-Npl residual refinement`

论文叙述口径：

- 仍然是统一输入体系
- 仍然是统一预测框架
- 全样本先经过同一个全局模型
- 仅在高 `Npl` 区间追加一个误差补偿分支

不建议表述为：

- “两个完全独立模型分别预测不同样本”

更适合表述为：

- “统一框架下的尾部增强”

## 3. 预期结构

### 阶段 A：全局主模型

- 输入：与当前最优单模型一致
- 输出：`Nexp (kN)` 的全局预测
- 配置基线：
  - `target_mode: raw`
  - `target_transform: log`
  - 保留数值几何特征

### 阶段 B：高 `Npl` 残差模型

- 训练目标：
  - `residual = y_true - y_global_pred`
- 训练样本：
  - 仅使用训练集中的高 `Npl` 样本
- 推荐首版阈值：
  - 使用训练集 `Npl` 的 `q5` 分界
  - 与当前 `regime_analysis.scale_npl_q5` 保持一致
- 推理逻辑：
  - 若样本落入高 `Npl` 区间，则最终预测为
    - `y_final = y_global + residual_correction`
  - 否则直接使用 `y_global`

## 4. 第一版实现原则

- 先做**硬阈值版本**
  - 简单、可解释、便于验证
- 不引入新的输入字段体系
  - 继续使用当前已有特征
- 阈值由训练集拟合
  - 推理阶段只复用阈值，不看测试集
- 残差模型先保持和全局模型同类
  - 仍用 `XGBoost`
- 第一版先不做软门控 / mixture 权重插值
  - 避免实现复杂度过早膨胀

## 5. 代码改造方向

### 5.1 元数据与工件

- 在训练元数据中记录：
  - 高 `Npl` 阈值
  - 全局模型参数
  - 残差模型参数
  - 尾部训练样本数量
  - 两阶段组合逻辑

### 5.2 训练流程

1. 按现有方式划分 train/test
2. 训练全局模型
3. 用训练集全量样本生成 `y_global_pred_train`
4. 在高 `Npl` 训练子集上构造残差标签
5. 训练残差模型
6. 在 train/test 上组合预测
7. 统一在 `Nexp` 空间汇报最终指标

### 5.3 预测流程

- 预测器需要支持：
  - 读取两阶段模型与阈值
  - 自动判断样本是否进入高 `Npl` 区间
  - 输出组合后的最终预测

## 6. 评估重点

第一优先：

- 总体 `R²`
- 总体 `COV`

第二优先：

- `scale_npl_q5` 的 `RMSE / R² / COV`
- `actual >= 3000 / 5000 / 10000 kN` 子集误差

第三优先：

- 是否损伤中低 `Npl` 区间表现
- 是否出现明显的尾部过补偿

## 7. 成功判据

最低成功标准：

- 相比当前最优单模型，总体 `R²` 提升
- 总体 `COV` 不劣化到 `0.1` 以上太多
- `scale_npl_q5` 指标明显改善

目标成功标准：

- 总体 `R² > 0.99`
- 总体 `COV < 0.1`

## 8. 风险

- 尾部样本仍然偏少，残差模型本身可能过拟合
- 硬阈值可能导致阈值附近预测不连续
- 若全局模型已经对尾部存在系统偏差，残差模型可能只是在“补锅”，需要严格对照验证

## 9. 若第一版失败的下一步

若硬阈值残差校正仍不足以过线，按以下顺序继续：

1. 把硬阈值改为平滑权重
2. 对残差模型单独做更小范围调参
3. 在全局模型中加入样本权重，和残差校正联合使用
4. 再考虑更复杂的门控或专家模型

## 10. 当前不做的事

- 不回到手工删更多数据作为主解法
- 不把截面类型拆成完全独立的主模型
- 不直接引入黑盒 stacking 作为论文主线
